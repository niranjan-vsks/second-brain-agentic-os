# sources/

Drop raw exports here, then run `directives/source_mining.md`:

- Call transcripts (discovery calls, client meetings, your own recorded 20-minute business monologue for the context files)
- Slack / Teams / Discord exports
- Email threads, old proposals, project retros
- Course/masterclass transcripts destined for skill bibles

Rules:

- Raw files stay raw — mining extracts notes, it never edits sources
- No credentials or client-restricted data (check `clients/{name}/rules.md` data boundaries first)
- After mining a file, prefix it with `mined_` so it isn't processed twice
